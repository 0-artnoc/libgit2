void git_object_free ( git_object *  object  ) 

Free a reference to one of the objects in the repostory.

Repository objects are managed automatically by the library, but this method can be used to force freeing one of the objects.

Careful: freeing objects in the middle of a repository traversal will most likely cause errors.

Parameters:
object the object to free

git_object* git_repository_lookup ( git_repository *  repo,
const git_oid *  id,
git_otype  type 
)

Lookup a reference to one of the objects in the repostory.

The generated reference is owned by the repository and should not be freed by the user. The generated reference should be cast back to the expected type; e.g.

git_commit *c = (git_commit *) git_repository_lookup(repo, id, GIT_OBJ_COMMIT);

The 'type' parameter must match the type of the object in the odb; the method will fail otherwise. The special value 'GIT_OBJ_ANY' may be passed to let the method guess the object's type.

Parameters:
repo the repository to look up the object
id the unique identifier for the object
type the type of the object
Returns:
a reference to the object
