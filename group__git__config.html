void *  data 
)

Perform an operation on each config variable.

The callback is passed a pointer to a config variable and the data pointer passed to this function. As soon as one of the callback functions returns something other than 0, this function returns that value.

Parameters:
cfgwhere to get the variables from
callbackthe function to call on each variable
datathe data to pass to the callback
Returns:
GIT_SUCCESS or the return value of the callback which didn't return 0
