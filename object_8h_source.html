00130 /**
00131  * Close an open object
00132  *
00133  * This method instructs the library to close an existing
00134  * object; note that git_objects are owned by the repository
00135  * and are reference counted, so the object may or may not be
00136  * freed after this library call, depending on whether any other 
00137  * objects still depend on it.
00138  *
00139  * IMPORTANT:
00140  * It is *not* necessary to call this method when you stop using
00141  * an object, since all object memory is automatically reclaimed
00142  * by the repository when it is freed.
00143  *
00144  * Forgetting to call `git_object_close` does not cause memory
00145  * leaks, but it's is recommended to close as soon as possible
00146  * the biggest objects (e.g. blobs) to prevent wasting memory
00147  * space.
00148  *
00149  * @param object the object to close
00150  */
